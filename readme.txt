This plugin changes the login logo for wordpress, removes admin corner logo, removes wordpress footer text, and disables admin widgets on the dashboard.

== Description ==

This Plugin is intended to be used to rebrand Wordpress with whatever business name or web developer information you choose.

A few notes about the sections above:

*Changes the login logo to a photo of your choosing.<br>
*Removes the Wordpress logo in the corner of the admin page.<br>
*Removes Wordpress footer text.<br>
*Disables most of the dashboard widgets.<br>

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Naviate to Settings->[Ctrl] Login to configure the plugin.
1. Use an uploaded photo from your Wordpress media section, copy the full http link and save it in the [Ctrl] Login media link input box then save changes.

== Frequently Asked Questions ==

= How do I use the plugin? =

Install plugin, on Admin panel go to settings -> [Ctrl] Login. Enter the web link copied from the media image on your site you want to use as your login logo.

= What size login logo is required? =

Logo will be automatically resized to 200px by 100px.

== Screenshots ==
1. Go to Media and find the logo you want to use or upload a logo
2. Copy media link
3. Go to admin settings then [Ctrl] Login and paste media link
4. Click save button
